# Askme - place where u can ask question

## Tech stack:

- Ruby on rails
- Html, css, erb
- PostreSql


Clone repository:
```bash
git clone https://github.com/SUppL7/AskFm.git
```
Install gems:
```bash
gem install 'devise'
gem install "font-awesome-rails"
require 'devise'
require "font-awesome-rails"
```

start project
``` bash
rails s
```
